# Cara install ?
Pastikan sudah terinstall nodeJS dan Git di komputer kalian.
- NodeJS : https://nodejs.org
- Git    : https://git-scm.com
- Git clone https://github.com/mhdnaufalhartg/SwiPop.git
- cd SwiPop
- npm install
- npm run dev
